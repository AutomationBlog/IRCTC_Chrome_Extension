# IRCTC 24hr to 12hr Time Converter

A simple Chrome extension that converts times on IRCTC website from 24-hour format to 12-hour format.

## Features

* Automatically converts times on IRCTC website to 12-hour format
* Option to toggle between 24-hour and 12-hour formats

## Installation

1. Download `IRCTC_24H_to_12H_Convertor.zip` file
2. Go to Chrome extensions page by typing `chrome://extensions/`
3. Enable Developer mode
4. Click on "Load unpacked"
5. Select the folder where you cloned the repository
6. The extension should now be visible in the extensions page

## Usage

1. Go to IRCTC website
2. The times should now be displayed in 12-hour format
3. To toggle between 24-hour and 12-hour formats, click on the extension icon and toggle the switch
